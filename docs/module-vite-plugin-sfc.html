Module: vite-plugin-sfc

Vite plugin that transforms SFC .html files into real ES modules at build time. Eliminates runtime string evaluation (AsyncFunction / unsafe-eval) by compiling each component's <template>, <style>, and <script> into a standard ES module during the Vite transform phase. The consuming app requires no special Content-Security-Policy directive. Each transformed .html module exports:
  • templateHtml — inner HTML of the <template> tag, or "".
  • styleText — inner text of the <style> tag, or "".
  • setup — async setup function receiving shadowDocument, or null.
Note: SFC root tags (<template>, <style>, <script>) must not carry HTML attributes.
